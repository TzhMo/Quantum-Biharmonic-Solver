# Quantum-Biharmonic-Solver
Classical and quantum simulation of Biharmonic equation

To evaluate the performance of the proposed methods, four separate programs were developed to solve the biharmonic equation under various boundary conditions, classified as follows:

Quantum Simulation Programs (Unitarylab Platform):

biharmonic_code.py : Solves the biharmonic equation with periodic boundary conditions and simply supported boundary conditions via quantum simulation.

Classical Verification Programs:

biharmonic_discretization.py: A conventional classical solver used to verify the numerical discretization accuracy for the Dirichlet_Neumann case.

splitting_method.py: A standard classical implementation dedicated to validating the splitting scheme and calculating the condition number of discretization system.
