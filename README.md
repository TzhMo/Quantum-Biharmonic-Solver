# Quantum-Biharmonic-Solver
Classical and quantum simulation of Biharmonic equation

To evaluate the performance of the proposed methods, four separate programs were developed to solve the biharmonic equation under various boundary conditions, classified as follows:

Quantum Simulation Programs (Unitarylab Platform):

[文件1名称].py : Solves the biharmonic equation with periodic boundary conditions via quantum simulation.

[文件2名称].py: Solves the biharmonic equation with simply supported boundary conditions via quantum simulation.

Classical Verification Programs:

[文件3名称].py: A conventional classical solver used to verify the numerical discretization accuracy for the Dirichlet_Neumann case.

[文件4名称].py: A standard classical implementation dedicated to validating the splitting scheme and calculating the condition number of discretization system.
